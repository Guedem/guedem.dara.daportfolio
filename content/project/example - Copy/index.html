This project is an exploration and analysis of the Craigslist advertising posts for housing rentals in the San Francisco Bay Area from 2000 to 2018. The dataset is publicly available on Kaggle. It was collected by economist Kate Pennington in 2018 using the Wayback Machine. It has more than 200,000 housing rental posts and seventeen variables, including date, price, number of bedrooms and bathrooms, etc. The project content goes as follow: an exploratory data analysis, data cleaning, descriptive statistics, regression analysis, a summary of the overall project, and finally citation of the sources.
